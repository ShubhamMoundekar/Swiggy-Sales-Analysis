# 📊 Swiggy Sales Analysis

## 📌 Project Overview
This project analyzes Swiggy sales data using Python to identify trends, patterns, and business insights.

---

## 🎯 Objectives
- Analyze sales performance
- Identify weekly trends
- Find peak order periods
- Compare category-wise revenue

---

## 🛠️ Tools & Technologies
- Python
- Pandas
- Matplotlib / Plotly
- Excel

---

## 📂 Dataset
- File: `swiggy_data.xlsx`
- Contains order details and revenue data

---

## 📊 Key Analysis
- Weekly Sales Trend Analysis
- Category-wise Revenue Analysis
- Order Distribution

---

## 🚀 How to Run
1. Download the repository
2. Keep `swiggy_data.xlsx` in same folder
3. Run:

```python
